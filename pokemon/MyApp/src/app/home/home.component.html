The Angular-SPARQL-Tutorial

This tutorial intends to enable you to build an easy Angular app, which can query a triple store (here: GrapDB), perform operations based on the retrieved data and to visualize this data to a user. Hereby, you will learn some basics on how to program a web-App with the Angular framework.

Lessons learned 1
How do I create, style and program an Angular app?

You will learn how to create an Angular App from scratch. Furthermore, you will learn how to use bootstrap in you App design in order to style your HTML in a very easy and efficient way. Lastly, you will learn how to create an interactiv app that is able to support a user based on the interaction with the app.

Lessons learned 2
How do I include RDF/OWL data from GraphDB in my app?

Build an interactive app is one thing, to build an Engineering tool another. Engineering tools have to have the ability to interact with Engineering or machine data. In this tutorial, you will learn how to query RDF/OWL data from a triple store and how to do operations on the returns. Furthermore, you will learn how to include this data in your app. Usually, the execution of queries is done by a backend. In order to provide a simple and easy start, the queries will be executed by Angular. However, this can be easily changed when switching to productive mode.
